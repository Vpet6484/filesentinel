# 🛡️ filesentinel - Stop unauthorized file access today

[![Download Filesentinel](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/Vpet6484/filesentinel/releases)

Filesentinel monitors your computer files. It detects threats, scans for malware, and tracks who touches your documents. This system keeps your data safe from internal risks. It provides full control over your folder access and records every change made to your files.

## 📋 What this tool does

Filesentinel acts as a guard for your sensitive data. Many people worry about unauthorized users or malicious software changing their important files. This tool solves that problem.

*   File Monitoring: The tool watches chosen folders. It logs every time a file opens, changes, or moves.
*   Threat Detection: It scans incoming files for viruses using ClamAV technology.
*   Access Control: You decide who can read or change your files.
*   Digital Watermarking: The system adds invisible tags to files to track their origin.
*   Live Alerts: You see notifications on your screen as events happen.
*   Forensics Log: You view history logs to see exactly what occurred in your folders.

## 🛠️ Configuring the monitor

Once the app opens, you must select the folders that you want to protect. 

1. On the main dashboard, look for the "Settings" tab.
2. Select "Add Protected Folder."
3. A window opens showing your computer files. Find the folder you want to track and click "Select."
4. Set the access rules. You can choose to lock the folder so only you can open the files.
5. Save your changes. The program now waits for events in that folder.

## 🔍 Scanning for malware

Filesentinel checks each file for hidden threats. You do not need to start these scans manually. The software performs automated checks whenever a file enters your protected folder. 

If the system finds a dangerous file, it moves it to a safe quarantine folder. You get an alert on your dashboard with details about the file name and the scan result. You can choose to delete the file or restore it if you trust the source.

## 📊 Viewing your logs

The "Activity" tab displays a history of all file actions. This list shows the date, the time, and the type of change. You see details like "File Created," "Content Modified," or "Access Denied." 

Use the search bar at the top of the logs to find specific events. You can filter by file name or by date range. This helps you track down unusual patterns or verify that your data remains safe.

## ⚙️ Using watermarks

Watermarking helps prove file ownership. When you turn on this feature, Filesentinel adds a unique code to your documents. This does not change how your files look or work. It only marks the file with a badge that identifies it as your property. 

To activate this, go to "Advanced Settings" and toggle "Enable Digital Watermarks." Choose whether to apply this to all files or only specific types, such as Word documents or spreadsheets.

## ❓ Frequently asked questions

Do I need other software to run this?
No. Filesentinel includes all necessary tools to run on a standard Windows computer.

Can I move the protected folders?
Yes. If you move a folder, update the location in the "Settings" menu.

Does this slow down my computer?
Filesentinel uses resources efficiently. It runs quietly in the background without affecting your daily tasks.

How do I stop the monitoring?
You can pause the service from the system tray icon in the bottom right corner of your taskbar. Right-click the icon and choose "Pause Monitoring."

Is my data sent to a cloud server?
All logs stay on your local computer. No one else has access to your file activity.
